NextCloud

发表于 2020-08-16 更新于 2022-08-30 分类于 工具配置 ， 自建网盘 Waline： 阅读次数： 本文字数： 896 阅读时长 ≈ 1 分钟

因为自己有一台阿里云 ECS，本着资源利用最大化的原则，想着搭建个私人网盘。

采用 Nextcloud，暂时搭建一个简易的先玩着：

https://segmentfault.com/a/1190000022620110

https://my.oschina.net/u/4339825/blog/3318845

自定义：

https://zimohan.com/it/nccustom.html

nextcloud 安装包地址：

https://download.nextcloud.com/server/releases/

备注：

最后一步时，提示采用的是 SQLite。

数据存放在服务器的哪里？

服务器的本地盘空间有限，也许需要再搭载个对象存储 OSS。

上传了各类资源测试了一下，感觉速度欠佳。

慢慢研究吧。

重启服务器后，通过 docker ps 查看，发现 nextcloud 未在运行。

docker ps：查看 nextcloud 是否在运行

docker pull nextcloud：拉取 nextcloud

错误：

内部服务器错误：解决办法 -》chown -R apache.apache /var/www/jhr/

创建管理员时提示用户已存在：可在 mysql 中通过 mysql> select distinct concat('user: ''',user,'''@''',host,''';') as query from mysql.user; 查看数据库的所有用户信息
