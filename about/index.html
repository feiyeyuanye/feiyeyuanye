关于博客

此博客主要当做笔记使用，也并没有特意收录到百度等，所以写的比较随意。

目前使用 Hexo Next 主题。

博客左上角书签，点击可记录浏览位置。

关于博客右下角的四个自定义按钮：

  1. 阅读模式：增加文章的可视阅读面积，尤其是移动端。
  2. 播放白噪音：现在只有一个下雨的声音。
  3. 列表切换：主页文章数大于 3 时，可以切换为两列显示。
  4. 换肤：更换文章背景色，普通/护眼/暗黑 三种模式切换。

待改善的问题：

  1. 博客内使用了大量的文字描述，这不利于理解，尤其是不利于记忆。鉴于能用图片表达时就不用文字表达，因此以后要注意多添加相应图像表达。
  2. 为了避免收藏的优质内容突然打不开，会将内容腾到这个博客上或是合并到相关文章上，所以导致一些博文的内容有重合的部分，或者是文章过长，待整合与抽离。
  3. 对于代码示例，尤其是上传到 GitHub 的项目练习，添加相应的图片或者是 git 图展示，便于以后查找需要的功能或样式。并且要添加项目描述，便于理解代码逻辑。
  4. 对描述性文字进行提炼，提炼出关键词或关键句进行展示，对详细解释进行折叠隐藏。
  5. 抄书式笔记要不得啊！即使站在巨人的肩膀上，也该有自己的思考与总结。
